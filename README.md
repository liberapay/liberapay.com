# Liberapay

[Liberapay](http://liberapay.com) is a recurrent donations platform. We help you fund the creators and projects you appreciate.

## Contributing

Liberapay is an open project, you can contribute to it. See [CONTRIBUTING.md](CONTRIBUTING.md) for instructions.

## Self-hosting

This webapp isn't self-hostable. Liberapay is a centralized platform that acts as a trusted third party, you can't host your own instance of it. What you can do is fork Liberapay, but you have to come up with a new name and logo, so you won't be self-hosting Liberapay.
